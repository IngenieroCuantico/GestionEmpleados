Comandos Generales:

1. creacion de proyecto laravel

composer create-project laravel/laravel gestionempleados

2. creacion de del middleware de autenticacion por laravel

php artisan make:auth

3. creacion de controladores 

php artisan make: controller Departamentos

4. creacion de modelos
 
php artisan make:model empleados


5. creacion de vistas

php artisan make:view empleados.index


6. creacion de funciones y recursos para un modelo-controlador

php artisan make:resource Departamento


7. para activar el servidor y poder visualizar los resultados en el localhost:8000

php artisan serve
